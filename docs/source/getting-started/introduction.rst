Introduction
============

A Shiny Application for Alternative Splicing Analysis and Peptide Mapping
--------------------------------------------------------------------------

IsoPepTracker is a bioinformatics tool designed to visualize, analyse and map peptides to genomic coordinates, and compare protein isoforms at the peptide level. It provides interactive visualizations to explore how alternative splicing influences protein diversity and peptide generation under various proteolytic digestion conditions.

**Web Portal**: https://www.isopeptracker.org
